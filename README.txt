Projet MAAIN 2019 Fontaine Victor

Le fichier frwiki-debut.xml contient les premières lignes du gros fichier xml (une centaine de pages seulement) pour pouvoir faire des tests rapide de l'application.
Le fichier frwiki-5000000l.xml contient les 5.000.000 premiere lignes du gros fichier xml et contient 7416 pages séléctionnées -> pour faire des tests plus intéréssants.

Pour compiler le programme : javac Main.java
Pour le lancer : java Main

Pour la premiere utilisation utiliser la méthode "init" pour créer les données nécessaire. A la fin de l'execution, ils seront stocké sous le format .data et pourrons etre réutilisés avec la méthode "read".
